# Expressing experimental designs using R formula

In this module, we will show how to use the two base R functions:

- `formula`
- `model.matrix`

...in order to produce design matrices for a variety of linear models.


```r
x <- c(1, 1, 2, 2)
f <- formula(~x)
f
```

```
## ~x
```



```r
model.matrix(f)
```

```
##   (Intercept) x
## 1           1 1
## 2           1 1
## 3           1 2
## 4           1 2
## attr(,"assign")
## [1] 0 1
```



```r
x <- factor(c(1, 1, 2, 2))
model.matrix(~x)
```
